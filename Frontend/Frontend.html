Editable td highlight clicked td content highlight searched keyword
ID Name Age
1 Alan 29
2 Banana 30
Description:
  1. 1. To make a td editable, set contenteditable="true" in a td.
  2. 2. To select clicked content, apply "selectText()" function.
  3. 3. To make the table sort by first column(ID) initially, set
    var table = $('#editableTable').DataTable({
    "order": [[1, "desc"]]
    });
  4. 4. To highlight the searched keyword, import jquery.highlight.js and set
    table.on('draw', function () {
    var body = $(table.table().body());
    body.unhighlight();
    body.highlight(table.search());
    });
    and CSS:
    table.dataTable span.highlight {
    background-color: rgb(255, 228, 74);
    border-radius: 0.28571429rem;
    }

    table.dataTable span.column_highlight {
    background-color: rgb(255, 228, 74);
    border-radius: 0.28571429rem;
    }
